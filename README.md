# Star Wars Page

Star Wars basic page using react router and data from an api


##Instal
```
    npm install

```

## Run
```
    npm  start
```

```
    yarn install
    yarn run start
```
Starts on `http://localhost:3000`

## Build
```
    npm run build
```

## Development technologies and external packages
* React 
* Jest & React testing library
* Styled Components
* React Router
* Axios
